Usage

Rename the file picture_fliper_config.json.sample.txt to picture_fliper_config.json 

Edit this file as folllows:

DIRECTORIES - comma separated list of directories containing pictures you want to display
EXTENSIONS - extensions of image types you want included
DELAY - milliseconds to display each image
TITLE - Text dispayed in the title bar of window
BOSS_FILE - absolute path of file to display when boss right click menu is selected


Known issues

Delay between pictures speeds up after a resume from a pause or boss menu selection
